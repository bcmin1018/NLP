리뷰를 4개의 점수로 분류하는 테스크
- 리뷰 점수 1,2,4,5 다중 분류 문제

시도한 점.
- 오버피팅 해결을 휘한 데이터 증강 기법을 사용 (Back Translation, Paragraph Generation, Random Swap, Random Deletion)
- 1단계 모델로 긍부정 이진 분류 모델, 2단계로 1,2점 4,5점 이진 분류를 위한 모델 2개 생성 

해결할 점.
- 데이터 증강을 하여 오버피팅을 막았으나 정확도가 떨어지는 것을 확인 할 수 있었다.
- test데이터 셋에 대한 정확도가 떨어지는 것은 데이터 증강의 품질이 나빠서 발생하는 것으로 추측되며 더 좋은 품질이 나올 수 있는 방안을 강구
- 계층 모델형 구현 미흡
